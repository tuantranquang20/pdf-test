# Elysia with Bun runtime

## RunDockerDB

```bash
 docker-compose up
```

## Install package
```bash
npm install
```

## Run prisma
```bash
npx prisma db push
npx prisma generate
```

## Development
To start the development server run:
```bash
bun run dev
```
